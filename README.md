# Raytracer

This project is a simple C++ raytracer and is an amalgamation of all of the research I have done on this topic.

# Light Types
The following light types have been implemented
- Ambient Lights
- Point Lights
- Directional Lights

# Lighting Features

The following lighting features have been implemented
- Diffuse Lighting
- Specular Reflections
- Reflections
- Shadows

# 3D Primitives
The following shape types have been implemented
- Sphere
