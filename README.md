# Raytracer

This project is a simple C++ raytracer and is an amalgamation of all of the research I have dont on this topic.

# Light Types
The following light types have been implemented
- Ambient Lights
- Point Lights
- Directional Lights

# Lighting Features

The following lighting features have been implemented
- Diffuse Lighting
- Specular Reflections
- Shadows

# Shapes
The following shape types have been implemented
- Sphere
